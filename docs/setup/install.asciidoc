[[install]]
== 安装 Kibana

NOTE: Starting with version 6.0.0, Kibana only supports 64 bit operating systems.

Kibana 提供以下格式的安装包 ：

[horizontal]
`tar.gz`/`zip`::

 `tar.gz` 包用来在 Linux 和 Darwin 系统下安装，而且也是开始 Kibana 最简单的一种选择 。
+
 `zip` 包是唯一支持 Windows 系统的安装包 。
+
<<targz>> 或者 <<windows>>

`deb`::

 `deb` 包用来在 Debian ，Ubuntu 和其他基于 Debian 的系统下安装 ，Debian 包可以从 Elastic 官网或者我们的 Debian 仓库中下载 。
+
<<deb>>

`rpm`::

 `rpm` 包用来在 Red Hat ，Centos ，SLES ，OpenSuSe 以及其他基于 RPM 的系统下安装 。RPM 包可以从 Elastic 官网或者我们的RPM仓库下载 。
+
<<rpm>>

`docker`::

一个镜像用于在 Docker 容器中运行 Kibana 。
https://www.elastic.co/products/x-pack[X-Pack] 预安装包附带 Kibana Docker 镜像，并且可以从 Elastic Docker 仓库中获得。
+
<<docker>>

IMPORTANT: If your Elasticsearch installation is protected by {xpack-ref}xpack-security.html[X-Pack Security]
see {xpack-ref}kibana.html[Using Kibana with X-Pack Security] for additional setup
instructions.
